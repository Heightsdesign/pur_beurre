READ ME 

Application Pur Beurre

Introduction

L'application Pur Beurre est une application qui vous permet de vous apporter une alternative
plus saine aux produits consommation quotidienne. 

1. Installation

Option 1

Téléchargez les documents du repositoire Code > Download ZIP puis décompressez le fichier ZIP.
Rendez vous dans le dossier "pur_beurre-master" et lancez la commande "pip install -r requirements.txt" 
afin d'installer tout les modules et librairies nécessaires au bon fonctionnement de l'application.

Option 2

Depuis votre terminal Gitbash créez un dossier à l'aide de la commande "mkdir <nom_de_votre_dossier>" 
puis téléchargez les fichiers de l'application "git clone https://github.com/Heightsdesign/pur_beurre.git".
Rendez vous dans le "pur_beurre" et lancez la commande "pip install -r requirements.txt" 
afin d'installer tout les modules et librairies nécessaires au bon fonctionnement de l'application.

2. Lancement de l'application

Se rendre dans le dossier de l'application via votre terminal puis
exécuter la commande "python3 -m main.py" afin de lancer l'application.

3. Fonctionnalités

Le menu d'accueil vous permet de vous rendre dans le menu "catégories", vous pourrez y sélectionner
la catégorie dans laquelle se trouve le produit que vous souhaitez substituer.
Vous pourrez également choisir de retrouver les "favoris" que vous avez enregistrez à partir de ce menu.

Si vous décidez de vous rendre dans les catégories, 
une liste de celle-ci vous sera alors proposée. Une fois la catégorie séléctionnée
vous pourrez alors choisir un produit a substituer parmi notre liste de produits.
Vous aurez ensuite la possibilité d'ajouter le résultat dans vos favoris.
